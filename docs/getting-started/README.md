# Getting Started with HealthSim Agent

Welcome to HealthSim! This guide will have you generating synthetic healthcare data in minutes.

---

## Prerequisites

Before you begin, ensure you have:

| Requirement | Minimum | Check |
|-------------|---------|-------|
| Python | 3.11+ | `python3 --version` |
| pip | Latest | `pip --version` |
| Anthropic API Key | - | [Get one here](https://console.anthropic.com/) |

---

## Quick Installation

```bash
# Install HealthSim
pip install healthsim-agent

# Set your API key
export ANTHROPIC_API_KEY="your-key-here"

# Start HealthSim
healthsim
```

That's it! You're ready to generate healthcare data.
