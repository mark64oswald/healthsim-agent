# Documentation Archive

This directory contains internal planning documents, design artifacts, and workspace-era documentation that has been archived to keep the main documentation clean and focused on end-user content.

**These documents are not part of the product documentation.**

---

## Directory Structure

### `planning/`

Internal planning and development documents created during the healthsim-agent development process:

- Implementation plans and progress tracking
- Gap analyses and assessments
- Design specifications and UX specs
- Super-prompts and session artifacts
- Migration planning documents

### `workspace-era/`

Documentation from the healthsim-workspace project that was copied during migration but is not applicable to the healthsim-agent architecture:

- MCP server documentation (agent doesn't use MCP)
- Workspace-specific guides
- Older format specifications

---

## Why Archive?

These documents were valuable during development but:
1. Are not relevant to end users
2. May contain outdated information
3. Would clutter the main documentation
4. Should not be confused with product docs
